Step 2: Enable Multi-Output

Once the plugin is loaded, enable multi-output mode. The process differs depending on your DAW.

Bitwig Studio Multi-Output Setup

In Bitwig Studio, click on the double arrows icon next to the plugin name and activate the missing chains.

Ableton Live Multi-Output Setup

Ableton Multi-Output Configuration

Ableton Live handles multi-output plugins differently than other DAWs. You'll need to manually route each output to separate audio tracks. OBSIDIAN Neural provides up to 9 individual stereo outputs (Track 1-8 + Main).

Important: Fix Plugin Resolution (Recommended First)
Prevent Blurry Plugin Interface

Ableton Live's auto-scaling can make OBSIDIAN Neural appear blurry. Fix this before loading the plugin by disabling auto-resize.

To get crisp, clear resolution for OBSIDIAN Neural:

  • In Ableton's left sidebar, locate the Plug-ins browser
  • Find OBSIDIAN-Neural in the plugin list
  • Right-click on OBSIDIAN-Neural
  • Uncheck "Auto-Scale Plug-In Window"
  • The plugin will now display at its native resolution without blurriness

Screenshot: Right-click menu showing "Auto-Scale Plug-In Window" option to uncheck

Why This Matters

Ableton's automatic window scaling can force plugins to resize, causing blurry text and controls. By disabling this option, OBSIDIAN Neural will always display at its designed 1300×800 resolution for optimal clarity.

Step 2: Create Audio Tracks for Each Output

Now create individual audio tracks to receive each track's output from OBSIDIAN Neural:

  • Create a new Audio track (this will be for Track 1)
  • In the "Audio From" section (top dropdown labeled "Input Type"), select the MIDI track containing OBSIDIAN Neural
  • In the lower "Audio From" dropdown (Input Channel), you'll see a list of available outputs:
    • Pre FX
    • Post FX
    • Post Mixer
    • Track 1-OBSIDIAN-Neural ← Select this
    • Track 2-OBSIDIAN-Neural
    • Track 3-OBSIDIAN-Neural
    • ... (up to Track 9)
  • Set the audio track's Monitor switch to "In" (this enables monitoring of the input)

Screenshot 2: Configuring audio track to receive Track 1 output

Important: Monitor Setting

Make sure to set Monitor to "In" on each audio track. Otherwise, you won't hear the output from OBSIDIAN Neural tracks.

Step 3: Repeat for Additional Tracks

Repeat the process for each OBSIDIAN Neural track you want to use:

  • Create another Audio track
  • Set "Audio From" to the OBSIDIAN Neural MIDI track
  • Select Track 2-OBSIDIAN-Neural (or 3, 4, etc.) from the Input Channel dropdown
  • Set Monitor to "In"
  • Repeat until you have all 8 tracks routed (or as many as you need)

Screenshot 3: Multiple audio tracks routing different OBSIDIAN Neural outputs

Step 4: Mute the Main MIDI Track

To avoid hearing the mixed output alongside individual tracks:

  • Mute or turn down the volume on the original MIDI track containing OBSIDIAN Neural
  • Now you'll only hear the individual track outputs through your audio tracks
Step 5: Add Effects to Individual Tracks

Now you can process each OBSIDIAN Neural track independently:

  • Add EQ, Reverb, Delay, Compression to each audio track
  • Adjust volume, pan, and send effects individually
  • Group tracks for easier organization
Pro Tip: Save as Template

Once you've set up all the routing and disabled auto-scaling, save your project as a template. This way, you can quickly start new projects with OBSIDIAN Neural already configured for multi-output with perfect resolution!

Step 5: Track Controls Overview

Each track in Obsidian Neural offers multiple interactive controls:

  • Show/Hide waveform
  • Show/Hide step sequencer (up to 4 bars)
  • Switch between original audio and SoundTouch stretched version
  • Generate new sample
  • Delete track
  • Beat repeat control
  • Enable/disable randomized quantization (1/2, 1/4, etc.)
  • Adjust quantization randomness manually with the knob
  • Pages A, B, C, D — each holding a different sample and prompt (e.g. A: “ambient guitar loop”, B: “industrial drum groove”)
  • Preview playback button per track

Step 6: Managing Prompts

On the main interface, you can add, edit, or delete prompts (except factory ones).
To edit or remove a prompt, Ctrl + right-click on the arrow in the prompt list.

When prompts are updated, all track prompt lists automatically refresh. You can also choose the scale/mode and sample duration (up to 30 seconds) before generating.

Step 7: Generate and Play Samples

To generate a sound on a track:

  • Select a prompt (either from the main list or the track prompt list)
  • Click Generate Loop on the bottom bar or the track’s generate button
  • Wait about 10 seconds for the AI to create your sample
  • Once ready, the waveform appears in the track view
  • Edit start and end loop points to isolate specific parts
  • Preview and toggle between original and stretched versions

Video: Generating and previewing samples in Obsidian Neural

Step 8: Sequencer and Mixer Controls

Use the step sequencer to trigger playback. Activate the first step, then arm the mixer channel’s Play button. Start your DAW — the sound plays on beat.

  • Adjust Pitch (-12 to +12 semitones)
  • Adjust Finetune (-50 to +50 cents)
  • Adjust Volume
  • Adjust Pan
  • All mixer parameters are MIDI-learnable (Pitch, Pan, Volume, Play, Mute, Solo)
Quantized Playback

Playback and sample loading are quantized — samples start exactly at bar 1 and step 0 for perfect synchronization.

UI Reset

If the interface becomes inconsistent, click the Reset UI button to restore the default layout.
