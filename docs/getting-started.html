Step 5: Install the VST Plugin

Extract and install the plugin to your VST3 folder so your DAW can detect it.

Windows Installation

Step 5.1: Extract the ZIP File
  • Navigate to your Downloads folder
  • Right-click on OBSIDIAN-Neural-Windows-VST3.zip
  • Select "Extract All..." or use your preferred extraction tool (7-Zip, WinRAR, etc.)
  • Extract to the same folder (Downloads)

Screenshot: ZIP file in Windows Explorer

Step 5.2: Locate the VST3 File

After extraction, navigate through the folders to find the actual VST3 plugin file:

C:\Users\YourName\Downloads\OBSIDIAN-Neural-Windows-VST3\OBSIDIAN-Neural.vst3\Contents\x86_64-win\
  • Open the extracted folder OBSIDIAN-Neural-Windows-VST3
  • Open OBSIDIAN-Neural.vst3 folder
  • Open Contents folder
  • Open x86_64-win folder
  • You should see the OBSIDIAN-Neural.vst3 file (not a folder, the actual VST3 file)

Screenshot: Navigating to the VST3 file in the extracted folder

Step 5.3: Copy to VST3 Folder

Now we need to copy the VST3 file to the standard VST3 installation directory:

Administrator Privileges Required

You may need administrator rights to copy files to the Program Files folder. If prompted, click "Yes" or "Continue" to allow the operation.

  • Right-click and copy the OBSIDIAN-Neural.vst3 file
  • Open File Explorer and navigate to:
C:\Program Files\Common Files\VST3
  • Right-click in the VST3 folder and select "Paste"
  • Click "Continue" or "Yes" if Windows asks for administrator permission
  • The installation is complete!

Screenshot: VST3 file copied to Program Files VST3 folder

Custom VST Folders

Some DAWs allow you to use custom VST folders. If your DAW is configured to scan a different location, you can copy the plugin there instead. Check your DAW's preferences for VST scan paths.

macOS Installation

Step 5.1: Extract the ZIP File
  • Navigate to your Downloads folder
  • Double-click on OBSIDIAN-Neural-macOS-VST3.zip to extract it
  • macOS will automatically extract the file, creating an OBSIDIAN-Neural.vst3 bundle
What is a .vst3 Bundle?

On macOS, OBSIDIAN-Neural.vst3 appears as a single item but is actually a folder (called a "bundle"). Don't try to open it - just copy/move it as-is!

Step 5.2: Install via Terminal (Recommended)

The easiest and safest way to install the plugin is using the Terminal with a simple command:

Administrator Password Required

You'll be prompted to enter your Mac's administrator password. This is normal and required to install plugins to the system folder. Your password won't be visible as you type - this is a security feature.

  • Open Terminal

    Go to Applications → Utilities → Terminal, or press Cmd + Space and type "Terminal"

  • Copy and paste this command into Terminal:
sudo cp -r ~/Downloads/OBSIDIAN-Neural.vst3 /Library/Audio/Plug-Ins/VST3/
  • Press Enter
  • Type your Mac password when prompted (it won't show as you type - this is normal)
  • Press Enter again
  • The plugin is now installed!
What This Command Does

This command copies the VST3 plugin bundle to the system-wide plugin folder where all your DAWs can find it. The sudo prefix gives the command administrator privileges needed to write to system folders.

Alternative: Manual Installation via Finder

If you prefer not to use Terminal, you can install manually:

  • Open a new Finder window
  • Press Cmd + Shift + G to open "Go to Folder"
  • Type or paste this path:
/Library/Audio/Plug-Ins/VST3/
  • Press Enter - the VST3 folder will open
  • Open another Finder window, navigate to Downloads, and find OBSIDIAN-Neural.vst3
  • Drag the OBSIDIAN-Neural.vst3 bundle into the VST3 folder
  • Enter your administrator password when prompted
  • Installation complete!
First Launch Security Verification

The first time you load OBSIDIAN Neural in your DAW, macOS Gatekeeper will verify the code signature:

✓ Code Signed & Notarized by Apple

OBSIDIAN Neural is officially signed and notarized by Apple. This means macOS trusts the plugin and you won't see security warnings or need to bypass Gatekeeper!

  • First Launch: macOS may take 5-10 seconds to verify the signature online (you'll see a brief progress dialog saying "Verifying...")
  • Subsequent Launches: The plugin will open instantly every time after the first verification
Internet Connection Required (First Launch Only)

macOS needs an internet connection the first time to verify the plugin signature with Apple's servers. After that initial verification, the plugin works offline (though you'll still need internet for AI sample generation).

User-Specific Installation (Optional)
Install for Current User Only

If you don't have administrator access or prefer a user-specific installation, you can install to your personal folder instead:

~/Library/Audio/Plug-Ins/VST3/

Terminal command for user installation:

cp -r ~/Downloads/OBSIDIAN-Neural.vst3 ~/Library/Audio/Plug-Ins/VST3/

Note: The tilde (~) represents your home folder. This method doesn't require sudo or administrator password.

Custom VST Folders

Some DAWs allow you to specify custom VST scan paths. If your DAW is configured to scan a different location, you can install the plugin there. Check your DAW's preferences under "Plug-ins" or "VST Settings".

Step 6: Configure the Plugin in Your DAW

Now let's add the Obsidian Neural plugin to your DAW and configure it with your API credentials.

Open Your DAW and Create a Project

  • Launch your DAW (Digital Audio Workstation)
  • Create a new project or open an existing one
  • If prompted, rescan your VST plugins to detect the newly installed Obsidian Neural

Screenshot: DAW project window

Add Obsidian Neural to an Instrument Track

  • Create a new Instrument Track in your DAW
  • Open the plugin browser and search for "Obsidian Neural"
  • Click or drag Obsidian Neural onto the instrument track
  • The plugin window should open automatically

Screenshot: Adding Obsidian Neural to an instrument track

Configure API Settings

To enable AI generation, you need to enter your API credentials from the dashboard:

  • In the Obsidian Neural plugin window, click the hamburger menu icon (three horizontal lines)
  • Select "Settings" or "API Configuration"

Screenshot: Obsidian Neural plugin with hamburger menu highlighted

Enter Your API Credentials
Where to Find Your Credentials

Go back to your dashboard at obsidian-neural.com and copy your API URL and API Key from the main page.

  • API URL Field:

    Paste the API URL from your dashboard (e.g., https://api.obsidian-neural.com/v1)

  • API Key Field:

    Paste your unique API Key from your dashboard

  • Click "Save" or "Apply" to save the settings

Screenshot: API settings dialog with fields for API URL and API Key
